# Support

Looking for help with a Mossworks Labs project? Pick the channel that matches what you need.

## Documentation

- **CRAFT Studio** — [docs.mossworks.io](https://docs.mossworks.io/) covers installation, architecture, and feature guides.
- **MCP servers** (`mcp-storytelling`, `mcp-comedy`, `mcp-yt-dlp`, `mcp-musicgen`) — each repo's `README.md` is the primary reference; the [Model Context Protocol docs](https://modelcontextprotocol.io/) cover client wiring.

## Asking a question

- Check the repo's existing [issues](https://github.com/orgs/Mossworks-Labs/repositories) and [discussions](https://github.com/orgs/Mossworks-Labs/discussions) first — the answer may already be there.
- Open a new issue using the **Question** option in the issue chooser. Include what you tried and what you expected.

## Reporting a bug

Use the **Bug report** template on the affected repo. See [`CONTRIBUTING.md`](CONTRIBUTING.md#how-to-file-an-issue) for what to include.

## Reporting a security issue

See [`SECURITY.md`](SECURITY.md). **Don't** open a public issue for security bugs.

## Commercial support

The platform projects (CRAFT, marketplace) are commercial offerings under licensed deployments. Contact us through the website at [mossworks.io](https://mossworks.io/) for licensing, hosted-instance pricing, or paid support engagements.
